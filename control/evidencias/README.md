# Evidencias por columna

Cada archivo de esta carpeta representa la evidencia operativa vigente de una columna del tablero.

## Regla de uso

1. Copiar la estructura de `_plantilla_evidencia_columna.md` si se necesita recrear un archivo.
2. Mantener un archivo por columna usando el nombre ya creado.
3. Actualizar `Resultado`, `Riesgo residual`, `Criterio de cierre a OK` y `Fecha` en cada iteracion.
4. Referenciar el archivo desde `control/tablero_mu_2026.tsv` y desde el reporte de fase.
