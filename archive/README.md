# Archivo Histórico del Repo

Este directorio separa del camino operativo principal el material `ARCHIVE` del inventario de limpieza MU 2026.

## Criterio

- Aquí viven artefactos con valor histórico, de contexto o de trazabilidad.
- No son la ruta oficial de ejecución ni de validación de MU Pregrado 2026.
- No deben borrarse por defecto.

## Subcarpetas

- `legacy_runtime/`: runtimes históricos no oficiales para MU 2026.
- `contexto_historico/`: decisiones, bitácoras y documentos técnicos históricos.
- `control_historico/`: guías de fase y plantillas ya reemplazadas por evidencia real.
- `resultados_legacy/`: salidas legacy o de otros flujos que no deben mezclarse con `resultados/` oficial.
- `resultados_remediacion/`: evidencia intermedia de la remediación ya cerrada.
- `resultados_diagnostico/`: ensayos, auditorías y stress tests históricos.
